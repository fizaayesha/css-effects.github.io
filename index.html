What is CSS?
​ ​ ​ ​ ​ ​ ​ ​CSS(Cascading Style Sheets), neither a programming language nor a markup language, but it's a STYLE SHEET LANGUAGE. CSS is an easy programming language to learn at a basic level. The CSS technology was designed to be accessible so anyone could create their own styled web pages on the internet. It is use to describe the presentation of a markup language. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files. Also, there are many advantages of CSS, like- it saves time, easy maintenance, gives superior styles to HTML, global web standardss, multiple device compatibility, pages load faster etc. There are basically three ways to give the CSS to the HTML text and theese are- Inline CSS, Internal or Embedded CSS and External CSS. I'll explain these three methods later. Also, we'll discuss many amazing properties of CSS and I will provide you separate code for HTML and CSS both.

Style Rules in CSS
​ ​ ​ ​ ​ ​ ​ ​A CSS consists of style rules that are integrated by browser and then applied to the corresponding elements in your document. A style role is made of three parts:
1) Selector: A selector is an HTML tag at which a style will be applied. This could be any tag like <p>, <span>, <h1>, <h2> etc.
2) Property: A property is a type of attribute of HTML tag. Put simply, all the HTML attributes are converted into CSS properties. They could be color, border, etc.
3) Value: Values are assigned to properties. For example, color property can have the value either red or #F1F1F1 etc.
